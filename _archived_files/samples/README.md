# samples/

このフォルダはローカル専用のサンプル/一時ファイル置き場です（Git にはコミットしません）。

想定パス例:
- `samples/json/recipes.json`
- `samples/json/recipes_converted.json`
- `samples/json/chiffon_recipes.json`
- `samples/pdfs/シフォンケーキ10選.pdf`

例:
- `node scripts/parse_chiffon_pdf.js`
- `node scripts/convert_units.js`

メモ:
- 生成物や取得したデータ（PDF/HTML/JSON/ログ等）は、著作権や機密の都合でリポジトリに入れない運用を推奨します。
