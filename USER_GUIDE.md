# レシピ管理アプリ 使い方ガイド

このアプリは、プロフェッショナルなレシピ管理のために設計されています。
日々の仕込み分量の計算、原価管理、多言語対応、そしてチームでのレシピ共有を安全に行うための機能が備わっています。

## 1. ログインとセキュリティ
- **ユーザーIDとパスワード**: 管理者から発行されたIDでログインします。
- **秘密の質問**: パスワードを忘れた場合に備えて、初回の登録時に「秘密の質問」を設定します。
- **最終ログイン**: 管理者は各ユーザーの最終アクセス日時を確認できます。

## 2. レシピ一覧と検索
- **検索**: 画面上部の検索バーで、レシピ名、材料、メモなどを横断検索できます。
- **フィルター**:
    - **店舗・コース・カテゴリー**: プルダウンで絞り込みが可能です。
    - **表示モード（通常/全表示）**:
        - **通常**: 一般的なレシピのみ表示します。
        - **全表示**: 隠しレシピや特殊なレシピも含めてすべて表示します。

## 3. レシピの閲覧・計算機能 (重要)
レシピ詳細画面では、仕込み量に応じた「自動計算」が可能です。

### パン・生地レシピ (Baker's %)
パンや発酵菓子などのレシピでは、**「狙いの総重量 (g)」** を入力するだけで、全ての材料（粉、水、イースト等）が自動的にベーカーズパーセントに基づいて再計算されます。
- 入力欄：画面上部の「総重量」に入力してください。

### 通常レシピ (倍率変更)
一般的な料理レシピでは、**「仕込み倍率」** を変更できます。
- **×1.0**: 元の分量
- **×2.0**: 2倍の分量
- **×0.5**: 半分の分量

### 翻訳機能
レシピは多言語に自動翻訳できます。
- 画面右上の言語切り替え（JA/EN/FR/ITなど）を選択すると、AIがその場で翻訳を表示します。
- 原文と比較したい場合は「原文を表示」スイッチをONにしてください。

## 4. レシピの作成と編集
### 新規作成
- **手動作成**: ゼロから入力します。
- **Web取り込み**: ネット上のレシピURLを貼り付けると、自動で解析して取り込みます。
- **画像取り込み**: レシピの写真をアップロードすると、AIが文字を読み取って入力します（手書きや写真の状態によっては修正が必要です）。

### 編集と安全性 (共有レシピの保護)
**「他人のレシピを間違って書き換えてしまう」心配はありません。**
- **自分以外のレシピ**（管理者や他のスタッフが作ったレシピ）を編集して保存すると、**自動的に「コピー」が作成されます。**
- 元のレシピはそのまま残り、あなたのリストに新しい編集版が追加されます。
- 自分のレシピであれば、そのまま上書き保存されます。

## 5. ゴミ箱と復元
- 不要なレシピは「削除」できます（ゴミ箱に入ります）。
- **ゴミ箱**: メニューから「ゴミ箱」を開くと、削除したレシピを確認・復元できます。
- **完全削除**: ゴミ箱の中でさらに削除すると、永久に消去されます。

<br>

## 6. 原価データの管理
仕入れリスト（CSVファイル）を取り込むことで、レシピ作成時の原価計算を自動化できます。

1. **データ管理画面**: メニューから「データ管理」を開きます。
2. **CSVアップロード**: お手持ちの原価リストをアップロードしてください。
    - **対応形式**: 「材料名, 単価, 単位, 業者」の並びのシンプルなCSV、または業務システムの出力形式に対応しています。
3. **自動入力**: レシピ編集画面で材料名を入力すると、登録された単価が自動的に反映されます。
