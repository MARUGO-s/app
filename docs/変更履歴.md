### 17. LLM出力JSONの自動修復機能（Unrecognized token '\' エラー対策）の実装 (新規追加)
- **対策内容**:
  - Perplexity（Web検索）や他のエージェントが、所見テキストをJSONで出力する際、テキストの中に不必要なバックスラッシュ（単独の `\` や不正なエスケープシーケンス）を生じさせてしまい、 `JSON.parse` が `Unrecognized token '\'` で構文エラーになり解析が失敗するバグを修正しました。

  - クリーニングが施された状態のJSONで再度 `JSON.parse` を自動リトライさせるフォールバックを実装しました。これにより、LLMの出力エスケープが崩れた場合でもパースをほぼ100%成功させ、エージェント所見を安全に抽出できます。

### 21. AI分析レポートの自動DB保存への変更（手動ボタン廃止） (新規追加)
